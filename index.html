Syntax

In an HTML document, raw texts are markup by an HTML element so computers can understand and render them correctly. Most of the elements are descriptive so it is very easy to learn and understand them. Every HTML element consists of an opening symbol '<' and a closing symbol '>'.

For example:

h1

The code above is how a heading can be defined in a document. h1 is the largest heading whereas h6 is the smallest. There is also a fixed set of tags that are used in every document.

The code is as follows:

!DOCTYPE html

Then followed by:

html

which is present twice in an HTML document, but with an additional symbol, like so:

/html

Not every element have an opening and closing tag. In this case, elements without a closing tag is a self-closing element. They are also called as void elements. One common element is the image element. They can be defined as: img /Note the additional forward slash symbol (/) is present in the end of the block.


Now that you know the very basic of how HTML documents are created, it is just a matter of knowing what elements to use. For more information and see the list of available elements, check the documentation here.
